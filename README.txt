This extension injects a dark themed stylesheet into pages (iframes) of a PDF within a PDF.js 
viewer hosted on a given website. This means it does NOT affect locally stored PDFs
being viewed in a browser. This also means that the extension requires access to all
websites as it can't predict which specific domains will be hosting a PDF.js viewer.

PDF Dark Reader - Manual Installation Guide
===========================================

--- GOOGLE CHROME ---

1. Go to https://github.com/ewnorman03/d2l-pdf-slides-darkreader in your browser.
2. Click the green "Code" button, then click "Download ZIP".
3. Once downloaded, extract the ZIP file to a folder of your choice.
4. Open Chrome and navigate to chrome://extensions in the address bar.
5. Enable "Developer mode" using the toggle in the top-right corner.
6. Click "Load unpacked".
7. In the file picker, select the "extension" folder from the extracted ZIP.
8. The PDF Dark Reader icon will appear in your Chrome toolbar.

To use:
- Navigate to any webpage with a PDF.js-based PDF viewer.
- Click the PDF Dark Reader icon in the toolbar.
- Click "Enable Dark Mode" to apply the dark theme to the current tab.
- Optionally, enable "Persistent Mode" to automatically apply dark mode
  to all PDF.js viewers you open going forward.


--- MICROSOFT EDGE ---

1. Go to https://github.com/ewnorman03/d2l-pdf-slides-darkreader in your browser.
2. Click the green "Code" button, then click "Download ZIP".
3. Once downloaded, extract the ZIP file to a folder of your choice.
4. Open Edge and navigate to edge://extensions in the address bar.
5. Enable "Developer mode" using the toggle in the left sidebar.
6. Click "Load unpacked".
7. In the file picker, select the "extension" folder from the extracted ZIP.
8. The PDF Dark Reader icon will appear in your Edge toolbar.

To use:
- Navigate to any webpage with a PDF.js-based PDF viewer.
- Click the PDF Dark Reader icon in the toolbar.
- Click "Enable Dark Mode" to apply the dark theme to the current tab.
- Optionally, enable "Persistent Mode" to automatically apply dark mode
  to all PDF.js viewers you open going forward.


--- MOZILLA FIREFOX ---

1. Go to https://github.com/ewnorman03/d2l-pdf-slides-darkreader in your browser.
2. Click the green "Code" button, then click "Download ZIP".
3. Once downloaded, extract the ZIP file to a folder of your choice.
4. Open Firefox and navigate to about:debugging in the address bar.
5. Click "This Firefox" in the left sidebar.
6. Click "Load Temporary Add-on...".
7. In the file picker, navigate to the "extension" folder from the extracted ZIP
   and select the "manifest.json" file.
8. The PDF Dark Reader icon will appear in your Firefox toolbar.

Note: Firefox temporary add-ons are removed when Firefox is closed. You will
need to reload the extension each time you restart Firefox.

To use:
- Navigate to any webpage with a PDF.js-based PDF viewer, or open a PDF
  directly in Firefox (Firefox's built-in PDF viewer uses PDF.js).
- Click the PDF Dark Reader icon in the toolbar.
- Click "Enable Dark Mode" to apply the dark theme to the current tab.
- Optionally, enable "Persistent Mode" to automatically apply dark mode
  to all PDF.js viewers you open going forward.
